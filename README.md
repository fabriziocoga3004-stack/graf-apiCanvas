## 2.2 Introducción a la graficación 2D 
## Autor: Alberto Fabrizio Cortes Garcia
## Fecha: 08/Marzo/2026
## Durante el desarrollo de la práctica se observó que el elemento Canvas permite un control preciso sobre la posición y el estilo de los elementos gráficos mediante el uso de coordenadas cartesianas. Asimismo, se identificó la importancia de organizar adecuadamente las figuras dentro del lienzo para evitar superposiciones y mantener una distribución visual equilibrada.
La utilización de métodos como arc(), moveTo(), lineTo() y quadraticCurveTo() permitió comprender cómo se generan formas más complejas a partir de primitivas gráficas básicas. Esto demuestra la versatilidad del elemento Canvas para la creación de interfaces visuales dinámicas dentro del entorno web.
